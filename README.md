# Bitcoin і BTC: економічний та статистичний аналіз

Проєкт створює відтворювану основу для економічного та статистичного аналізу Bitcoin і BTC.

Поточний статус: нульовий етап — технічна основа.

## Передумови

Потрібні R, RStudio, Quarto та Git.

## Відтворення

У корені проєкту виконайте:

```r
renv::restore()
```

Потім побудуйте книгу:

```text
quarto render
```

Згенерований HTML-файл буде розташований у `_book/index.html`.
